An easy way to add Pinterest pins slider on your WordPress website as a widget.

== Description ==

This is an early version of the Slider Widget For Pinterest Pinboard for WordPress, please let me know at http://akashsoni.com if you have suggestion how to make it better.

The plugin create a slider of your pinterest pins on WordPress site.

	1 ) You need to enter your pinterest username and boradname in widget. if you not enter any boardname then it show latest pins in slider. 

	2 ) you can set color of slider arrows.

	3 ) limitation of pins.

	4 ) Set slider animation.

	5 ) Slider #number of columns.

This plugin feched maximum 50 pins from your pinterest account.

This plugin is maintained by Akash soni, http://akashsoni.com 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/slider-widget-for-pinterest-pinboad` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go To Appearance > Widgets  add Pinterest Slider in your widget area and add all details in Pinterest Slider widget.
